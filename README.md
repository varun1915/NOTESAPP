# NOTESAPP
This a Node Application to add, remove and list notes.

Go to master branch




# Get Started
Clone this Repository on you machine locally.



## Add a Note
`node index.js add --title="" --body=""`

## Remove a Note
`node index.js remove --title=""`

## List all Notes
`node index.js list`

## Read a Note
`node index.js read --title=""`
